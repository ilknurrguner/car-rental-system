# Car Rental System (Python)

## Description
A desktop-based car rental application developed using Python and Tkinter.
The application allows users to view vehicle information, manage rental status,
and track availability through a graphical user interface.

## Team Project
This project was developed as a **team project** during the semester.
All team members contributed equally to the development.

## Technologies Used
- Python
- Tkinter
- JSON

## Features
- List available, rented and maintenance vehicles
- Rent and return vehicles
- Store vehicle and rental data in JSON files
- Graphical interface with action buttons

## Project Files
- `arayuz.py` – Graphical user interface
- `fonksiyonlar.py` – Application logic and functions
- `aracListesi.json` – Vehicle data
- `modelListesi.json` – Vehicle models
- `temp_sirali.json` – Temporary sorted vehicle data
- `delete.png`, `edit.png` – UI assets
